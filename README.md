# Jungle Tank

탑뷰 시점에서 탱크를 조작해 전투하는 웹 게임 프로젝트입니다.  
현재 버전은 팀전 봇전 기반의 MVP이며, 이후 멀티플레이와 전장 확장을 고려한 방향으로 발전시키는 것을 목표로 합니다.

## 현재 구현 상태

- HTML/CSS/Vanilla JS Canvas 기반 탑뷰 탱크 전투
- 플레이어 1명 + 아군 봇 vs 적 봇의 팀전
- 메뉴에서 아군/적군 수를 조절하는 `N vs M` 설정
- 이동, 조준, 발사, HP, 리스폰, 점수 시스템
- 팀 단위 점수, 리스폰, 승리 조건
- 파워업 3종
  - 회복
  - 연사 강화
  - 1회 보호막
- 팀 규모에 따라 달라지는 선승 점수 규칙
- 결과 화면 및 재시작

## 실행 방법

```bash
npm install
npm start
```

브라우저에서 `http://127.0.0.1:4173` 으로 접속하면 플레이할 수 있습니다.

검증 스크립트 실행:

```bash
npm run verify
```

## 조작법

- 이동: `WASD` 또는 방향키
- 조준: 마우스
- 발사: 마우스 클릭 또는 `Space`
- 팀 수 조절: 메뉴에서 `+`, `-` 버튼 클릭
- 전체화면: `F`

## 추가 예정 기능 명세

### 1. 음악 + 효과음

- 배경 음악(BGM) 재생
- 발사, 피격, 폭발, 파워업 획득 등 주요 이벤트에 효과음 추가
- 게임 상태에 따라 사운드 on/off 또는 볼륨 조절 기능 고려

### 2. 여러 맵 지원

- 단일 고정 맵에서 벗어나 복수 전장 선택 가능 구조로 확장
- 맵마다 장애물 배치, 엄폐 구조, 스폰 위치를 다르게 구성
- 향후 맵 선택 화면 또는 랜덤 맵 로테이션으로 확장 가능

### 3. 팀전 고도화

- 현재는 플레이어 1명 + 아군 봇 / 적 봇 구조의 `N vs M` 팀전을 지원
- 향후에는 역할군 분리, 분대 행동, 팀별 전략 차별화까지 확장 가능
- 예시
  - 2 vs 2
  - 3 vs 3
  - 4 vs 2 같은 비대칭 구성

### 4. 지뢰 설치 기능

- 탱크가 맵에 지뢰를 설치할 수 있는 기능 추가
- 지뢰는 일정 시간 유지되거나, 적이 접근하면 폭발
- 범위 피해, 시각 효과, 설치 개수 제한 등의 규칙 정의 필요
- 파워업/맵 구조와 연계해 전략 요소로 확장 가능

## 프로젝트 방향

현재는 팀전 봇전 MVP이지만, 이후에는 다음 방향으로 확장하는 것을 전제로 합니다.

- 사운드 연출 강화
- 맵 다양화
- 팀 기반 전투 고도화
- 설치형 무기와 전략 요소 추가
- 멀티플레이 구조 확장 가능성 검토

## 협업 최소 규칙

- 한 번에 하나의 목적만 수정한다. 기능 추가, 버그 수정, 리팩터링을 한 커밋에 섞지 않는다.
- 작업 전 현재 브랜치와 변경 파일을 확인한다.
- 동작이 바뀌면 직접 실행해서 확인하고, 가능하면 `npm run verify`까지 통과시킨다.
- 큰 구조 변경 전에는 README나 이슈에 의도를 먼저 남긴다.
- 사용하지 않는 코드나 임시 로그는 커밋 전에 제거한다.
- `node_modules`, 테스트 산출물, 개인 로컬 파일은 커밋하지 않는다.
- 새 기능을 넣으면 README의 기능 설명이나 명세도 같이 갱신한다.
- 다른 사람이 작업 중인 파일은 함부로 덮어쓰지 말고, 충돌 가능성이 있으면 먼저 공유한다.
