# Tank Audio Pack

`bgm/tank_battle_bgm.wav`
- Original action BGM with fast tempo and arcade shooter energy.
- Duration: about 26.75 seconds. Loop this in-game.

`effects/player_shot.wav`
- Player basic cannon shot.

`effects/enemy_shot.wav`
- Enemy basic cannon shot with a slightly heavier, rougher tone.

`effects/tank_hit.wav`
- Tank hit / damage impact.

`effects/shield_pickup.wav`
- Shield activation pickup.

`effects/heal_pickup.wav`
- Heal pickup.

`effects/power_pickup.wav`
- Power-up pickup cue.

`effects/power_cannon_shot.wav`
- Use this instead of the normal shot while the power-up is active.
- Intended active duration: 3 seconds.

## Regenerate

Run:

```bash
python3 sound/generate_audio.py
```
